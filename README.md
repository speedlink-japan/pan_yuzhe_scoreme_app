# 🎀 MyScore - Self-Development Game

自分自身を育成するゲーム型の自己分析・習慣作り・作業サポートツール

## 概要

MyScore は、キャラクター作成から着せ替え、ポイント稼ぎ、報酬交換まで、ゲーム要素を取り入れた自己育成ツールです。

- **自己分析**: キャラクターを通じた自分を表現
- **習慣作り**: TODO、読書、メモ記録でポイント獲得
- **作業サポート**: ポイントに基づく報酬システムで動機付け
- **ゲーム性**: キャラクター育成、着せ替え、インタラクション

## 主要機能

### 1️⃣ **選択バー** (画面下部)
4つの主要機能を自由に表示・非表示

- **TODO** (✓) - タスク管理
- **CALENDAR** (📅) - カレンダー表示
- **NOTEBOOK** (📝) - ノート・メモ・読書記録
- **ME** (🏠) - キャラクター育成・着せ替え

### 2️⃣ **TODO 機能**
- タスク追加・チェック
- 完了ごとにポイント自動加算
- 画面: チェックマーク系UI

### 3️⃣ **CALENDAR 機能**
- 月間カレンダー表示
- 当日ハイライト
- 見た目キューティ重視

### 4️⃣ **NOTEBOOK 機能** (メイン)

複数の記録方法をサポート：

#### 📕 本を読む
- PDF・テキストファイル追加
- ページ数で自動ポイント計算
- 読んだページ数のトラッキング

#### 📝 メモを書く
- 色選択可能なメモ用紙
- 記録文字数でポイント転換
- テンプレート機能

#### 📖 ノートを読み込む
- 既存ノートを読み込んでポイント計算
- **メイン機能として目立たせる**
- 過去の記録から学習・復習

### 5️⃣ **ME 機能** (メイン・ゲームコア)

#### キャラクター育成
- アプリ起動時にキャラクター作成画面を全表示
- "今日の私はどう？" というテーマ文字
- 開くたびに着せ替え可能

#### ゲーム要素
3つのインタラクションボタン：

- **☕ 一息**: キャラとお茶・お菓子を一緒に楽しむ
- **✨ おしゃれ**: 着せ替え機能
- **🎲 趣味**: 背景のものにアクセス、アニメーション＋特殊セリフ表示

#### 報酬システム
- ポイント蓄積
- 着せ替えアイテム、飾り、ホーム装飾と交換
- キャラクターレベル上昇

## システムの流れ

```
✨ ポイント獲得 (TODO完了、本読書、メモ記録)
↓
💰 ポイント蓄積
↓
🎁 報酬交換 (着せ替え、ホーム装飾など)
↓
🎀 キャラクター育成・メイン機能アンロック
```

## 技術スタック

- **Frontend**: Next.js 15 + TypeScript + React 18
- **Styling**: Tailwind CSS + CSS Modules
- **Color Scheme**: 青系と淡い黄色（すっきりな雰囲気重視）

## インストール & 実行

### 必須要件
- Node.js 18 以上
- npm または yarn

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開く

### プロダクションビルド

```bash
npm run build
npm start
```
